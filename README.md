# HackathonLowCode - Filtro de dados dinâmico

Esse projeto foi desenvolvido durante a Hackathon da Rerum durante a XXVI Semana da Computação da Universidade Federal de Juiz de Fora (UFJF) como solução para o tema 1, LowCode e geração de um formulário genérico para edição de registros de uma tabela qualquer.

A ferramenta diminui a necessidade de intervenção humana em processos de consolidação de dados para transformá-los em informação útil. Para aumentar a produtividade no desenvolvimento de formulários de cadastramento, a aplicação apresenta uma solução rápida e dinâmica para esse processo de validação.


## Tecnologias utilizadas

**Front-end:** HTML, CSS, Bootstrap e Js

**Back-end:** Python com Flask

**Database:** SQLite


## Instalação

Dependências do projeto:

```
  py -m pip install flask
  py -m pip install pandas
  py -m pip install numpy
  py -m pip install flask_core
  py -m pip install os
```

Use a Extensão "Live Server (Five Server)" do VSCode para criar um servidor para testar esse projeto.
Execute o Backend.py e depois abra os .html com o Live Server.
